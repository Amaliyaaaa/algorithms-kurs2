# algorithms-kurs2
Целью работы является разработка приложения для решения задачи о рюкзаке с использованием метода динамического программирования. Программа реализована в среде MS Visual Studio 2022 на языке программирования C#, с использованием технологии Windows Forms для создания пользовательского интерфейса.

Объектом исследования является задача о рюкзаке, которая является классическим примером задачи оптимизации, широко используемой в различных областях, таких как логистика, распределение ресурсов и финансовое планирование.
Предметом исследования являются методы динамического программирования для решения задач оптимизации с ограничениями и их применение в разработке программных решений на платформе .NET.

В ходе работы была разработана программа, позволяющая решать задачу о рюкзаке с учетом ограничений по весу. Пользователи могут вводить данные о предметах через графический интерфейс, а алгоритм динамического программирования находит оптимальное решение задачи, выводя на экран максимальную ценность и список выбранных предметов. Время работы алгоритма также отображается для оценки его производительности.

Задача о рюкзаке является одной из классических и наиболее известных задач в области теории алгоритмов и структур данных. Она представляет собой задачу оптимизации, где необходимо выбрать подмножество предметов с заданными весами и ценностями так, чтобы их общая ценность была максимальной, а общий вес не превышал определённой вместимости рюкзака. Эта задача имеет широкое применение в различных областях, таких как логистика, планирование, финансовое моделирование, а также в задачах оптимизации с ограничениями.

## Что реализовано:
В главном окне реализован весь функционал приложения при помощи таких интерактивных элементов, как текстовые поля для ввода или выбора значений количества предметов и вместимости рюкзака, кнопки для генерации списка предметов и для решения задачи, а также поля для вывода конечного оптимизированного списка предметов.

Статическим классом я реализовала фиксированный список из 50 предметов, которые придумала сама. Есть возможность протестировать алгоритм на большем количестве предметов со случайными значениями веса и ценности - до 10 000. Вместимость рюкзака - до 1000.

При решении задачи выводится время выполнения программы и объем используемой памяти для выбранных пользователем параметров (описаны выше).
